Random Planer

작심삼일. 이제는 RPlaner와 함께하세요.
RPlaner는 사용자가 생성한 계획중에 랜덤하게 계획을 뽑음으로써, 사용자에게 재미있게 계획을 실천할 수 있도록 도와줍니다.
카운트다운과 그래프를 활용하여 사용자에게 시각적으로 계획을 실천할 수 있도록 압박을 주게됩니다. RPlaner.

추가 기능 구현 예정

원하는 메세지 알림 기능

계획 생성 시, 원하는 시간에 왔으면 하는 메세지를 적어, 사용자가 지정한 데드라인 동안 매일 지정한 시간에 알림이 가도록 한다.

구글 계정과 연동

OAuth를 이용하여 구글 계정으로 로그인하여, 내가 생성한 계획이 캘린더에도 표시되게 한다.

Priorty 추가

계획 생성시, 중요도를 주며 계획 리스트에서 색깔로 구별 할 수 있게 한다. 또한 중요도별로 정렬할 수 있게 한다.

랜덤 계획을 뽑는 알고리즘 개선

중요도가 높은 것 부터 뽑혀야 하므로 랜덤 계획을 뽑는 알고리즘을 개선하여 중요도가 높은 것보다 낮을 것이 먼저 뽑히지 않도록 한다.
