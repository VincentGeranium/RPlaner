RPlaner

작심삼일. RPlaner와 함께라면 알차게 보낼 수 있습니다.
RPlaner 보러가기.

작심삼일이 모이면 1년에 약 122개의 계획들을 실천할 수 있습니다.
RPlaner는 사용자가 생성산 계획중에 랜덤한 계획을 뽑음으로서, 사용자에게 재미있게 계획을 실천할 수 있도록 도와줍니다.
또한 카운트다운과 그래프를 활용하여 남은 시간을 나타내주므로 사용자에게 남은시간을 한눈에 알아볼 수 있도록 해줍니다.

Used technology

RPlaner는 다음과 같은 기술들로 이루어져있습니다.

  • Swift
  • Realm
  • KDCircularProgress
  • User Notifications
  • GameplayKit

RPlaner Actual screen

간편한 계획생성

구체적인 계획을 설정하고 1일부터 7일까지 원하는 일 수를 데드라인으로 줄 수 있습니다.
또한 사용자가 하려는 계획이 한 번만 수행하면 되는 소모성 계획이라면 '이내'를,
기한 내내 해야하는 계획이라면 '동안'을 줄 수 있습니다.

클릭 한번으로 랜덤한 계획을..

클릭을 누르게 되면 사용자가 생성한 계획들 중 랜덤한 계획이 나오게 됩니다.

카운트다운과 그래프

시각적으로 사용자에게 남은 시간을 글과 그래프로 보여줌으로서 사용자의 계획 실천을 돕습니다.

카운트 다운이 끝나면..

User Notifications을 이용하여 사용자에게 알림이 가게 됩니다.
